Tentang Saya

Halo! Saya adalah seorang pengembang web yang bersemangat untuk menciptakan pengalaman digital yang menarik dan bermanfaat. Saya memiliki latar belakang dalam teknologi informasi dan telah bekerja pada berbagai proyek pengembangan web.

Proyek

Proyek 1: Website Perusahaan

Proyek ini adalah website perusahaan untuk perusahaan fiktif, dibangun dengan HTML, CSS, dan JavaScript. Website ini menampilkan informasi tentang perusahaan dan layanan yang ditawarkan.

Proyek 2: Aplikasi Mobile

Aplikasi mobile ini dibuat menggunakan React Native dan menyediakan fitur pencarian restoran terdekat berdasarkan lokasi pengguna. Pengguna juga dapat melihat menu dan membuat reservasi melalui aplikasi ini.
